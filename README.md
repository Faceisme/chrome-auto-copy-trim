# Auto Copy Trim

Chrome 扩展：选中文字后自动复制，并在复制前去掉两边的空格、换行符、制表符等空白字符。复制成功会在页面上显示一个小提示：`copied`。

图标是一组内置 PNG：深色圆角底、剪贴板主体和绿色复制状态点，适配 Chrome 工具栏和扩展管理页。

## 默认设置

- 普通网页选中后自动复制：开启
- 输入框、文本框、可编辑区域：按住 Ctrl 选中时自动复制：开启
- 鼠标中键粘贴：开启
- 粘贴时覆盖原内容：开启（关闭后改为在光标位置插入）

这些设置可以点击 Chrome 工具栏里的扩展图标修改。

## 安装

1. 打开 Chrome：`chrome://extensions/`
2. 右上角打开“开发者模式”
3. 点击“加载已解压的扩展程序”
4. 选择这个文件夹：
   `C:\Users\merci\Documents\Codex\2026-06-17\chrome-copy\outputs\chrome-auto-copy-trim`
5. 已打开的网页刷新一次后生效

如果之前已经安装过这个扩展，在 `chrome://extensions/` 里点扩展卡片上的刷新按钮，然后刷新网页。

## 使用

- 普通网页正文：直接选中文字，会自动复制。
- 输入框、textarea、contenteditable：按住 Ctrl 选中文字，才会自动复制。
- 鼠标中键粘贴：在输入框、textarea、contenteditable 中点击鼠标中键。默认会清空原内容再粘贴（覆盖）；如果想保留原内容、只在光标位置插入，可在扩展弹窗里关闭“粘贴时覆盖原内容”。

## 限制

- Chrome 内部页面无法注入扩展脚本，例如 `chrome://extensions/`、Chrome 网上应用店。
- 部分网页使用很重的自定义编辑器，可能会拦截中键或剪贴板事件；扩展已经做了剪贴板读取兜底，但网页自身限制仍可能影响粘贴位置。
- 如果要在本地 `file://` 页面使用，需要在扩展详情页打开“允许访问文件网址”。
